howsun-javaee-framework
=======================

Java应用层框架

版本：1.8

这是一款特别适用于中小企业应用的JavaEE快速开发框架。它是居于Spring容器之上，封装了DAO（含Hibernate和MongoDB）操作、多模块统一管理、统一配置管理、统一日志管理等优雅的工程管理开发模型，并提供大量工具包、Json操作、分页辅助工具。
